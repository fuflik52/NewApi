# RUST UI DESIGN SYSTEM (SVG STYLE GUIDE)

Этот документ описывает визуальный стиль для генерации SVG интерфейсов на основе проанализированных скриншотов (Battle Pass, Main Menu, Settings, Shop).

## 1. Цветовая палитра (Color Palette)

### Основные цвета (Primary)
*   **Акцентный Синий (Highlight/Active):** `#3B99FC` или `rgba(59, 153, 252, 1)`
    *   *Использование:* Активные пункты меню, кнопки настроек, прогресс-бары, выделение.
*   **Акцентный Зеленый (Success/Action):** `#7AB55C` или `rgba(122, 181, 92, 1)`
    *   *Использование:* Кнопки "ЗАБРАТЬ", "ДОСТУПНЫЕ КИТЫ", индикаторы выполнения, уровни.
*   **Акцентный Красный (Danger/Blocked):** `#D9534F` или `rgba(217, 83, 79, 1)`
    *   *Использование:* "БЛОКИРОВКА ОРУЖИЯ", недоступные элементы.
*   **Золотой/Желтый (Premium/Gold):** `#F0AD4E`
    *   *Использование:* Премиум статус, VIP элементы.

### Фоны и Подложки (Backgrounds & Opacity)
*   **Глобальный оверлей (Global Overlay):** `rgba(15, 15, 15, 0.95)` - Почти черный, сильно затемняет игру.
*   **Фон Сайдбара (Sidebar BG):** `rgba(20, 20, 20, 0.9)` - Темная полоса слева.
*   **Фон Карточек/Панелей (Card BG):** `rgba(255, 255, 255, 0.05)` или `rgba(40, 40, 40, 0.6)` - Светлее фона, чтобы выделить контент.
*   **Фон Слотов (Slot BG):** `rgba(0, 0, 0, 0.5)` - Темные квадраты под иконки предметов.

### Текст (Typography Colors)
*   **Основной текст (Primary Text):** `#FFFFFF` (Белый) - Заголовки, активные элементы.
*   **Вторичный текст (Secondary Text):** `#AAAAAA` (Светло-серый) - Описания, неактивные пункты меню, таймеры.
*   **Текст на кнопках:** Обычно белый `#FFFFFF`.

## 2. Структура и Макет (Layout & Shapes)

### Сайдбар (Sidebar)
*   **Расположение:** Слева, фиксированная ширина (примерно 250-300px).
*   **Элементы:** Иконка слева + Текст справа (Uppercase).
*   **Активное состояние:**
    *   Синяя полоска слева (border-left: 4px solid #3B99FC).
    *   Либо фон кнопки становится светлее/синее `rgba(59, 153, 252, 0.2)`.

### Кнопки (Buttons)
*   **Форма:** Прямоугольные.
*   **Скругление (Border Radius):** Минимальное (0px - 2px). Стиль "строгий/квадратный".
*   **Размеры:** Высота обычно 40-50px.

### Карточки и Сетки (Cards & Grids)
*   **Сетки предметов:** Квадратные ячейки с небольшим отступом (gap).
*   **Бордеры (Borders):**
    *   Обычно отсутствуют (используется разница в цвете фона).
    *   Активный элемент может иметь обводку `stroke="#3B99FC" stroke-width="2"`.

## 3. Типографика (Typography)
*   **Шрифт:** Sans-serif (Roboto, Open Sans или стандартный системный).
*   **Регистр:**
    *   Заголовки разделов: **UPPERCASE** (Все заглавные).
    *   Пункты меню: **UPPERCASE**.
    *   Описание: Sentence case (Обычный).
*   **Вес (Weight):**
    *   Заголовки: Bold (700).
    *   Текст: Regular (400).

## 4. Специфические Элементы (Specific UI Elements)

### Battle Pass / Прогресс
*   **Линия прогресса:** Фон темно-серый, заполнение зеленое (`#7AB55C`) или желтое для премиума.
*   **Уровни:** Квадратные плашки с цифрами.

### Блокировка оружия (Wipe Block)
*   **Слот:** Затемненная иконка оружия.
*   **Таймер:** Текст поверх слота или полоска с надписью "1 ЧАС...".

### Настройки (Settings)
*   **Переключатели:** Квадратные кнопки выбора (не радио-кнопки, а плитки). Активная плитка подсвечена цветом и галочкой.

## 5. SVG Технические требования
*   Использовать `fill-opacity` вместо `opacity` для тонкой настройки прозрачности цвета.
*   Шрифты переводить в кривые не обязательно, если используется стандартный шрифт, но для гарантии отображения лучше использовать системные шрифты.
*   Иконки: Flat стиль, векторные, одноцветные (белые или цветные в зависимости от редкости).
